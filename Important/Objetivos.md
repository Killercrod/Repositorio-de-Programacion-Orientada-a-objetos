# Objetivos Logrados:
 1. Se creo un algoritmo funcional para la detección de los datos de Nombres y CURPS.
 2. Se hizo una base de datos que almacena todos los datos de los usuarios que se registran.
 3. Se realizo un método funcional para detectar errores ingresados en un CURP.
 4. Se crearon métodos para corregir tanto datos mal capturados por el programa como mal ingresados por el usuario.
 5. Trasladar el proyecto de computadoras a móviles.

# Objetivos no Logrados
1. No se incluyo un sistema para cargar archivos de INEs escaneadas.
2. No se incluyo una función para tomar la foto automaticamente.
